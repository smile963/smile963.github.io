隐写工具outguess安装使用介绍

发表于 2020-03-23 | 分类于 工具

outguess下载安装

kail 终端命令输入git clone https://github.com/crorvick/outguess，安装包下载完成到文件夹。打开文件夹，右键空白处选终端打开，输入命令./configure && make && make install进行安装。

使用outguess

输入outguess -help即可获得相关命令。
加密：
outguess -k “my secret key” -d hidden.txt demo.jpg out.jpg
加密之后，demo.jpg会覆盖out.jpg，hidden.txt的内容是要隐藏的东西。
解密：
outguess -k “my secret key” -r out.jpg hidden.txt
解密之后，紧密内容放在hidden.txt中

BUUCTF 2nd-Ctypto 老文盲了

发表于 2020-03-23 | 分类于 CTF

题目

里面是一个txt文件，内容为:

罼雧締眔擴灝淛匶襫黼瀬鎶軄鶛驕鳓哵眔鞹鰝

思路

由于做Ctypto题不久，看到这第一时间找这是什么加密方式，找了半天找到了与佛论禅加密中的字体和这个挺像的，然而这种加密需要“佛曰：”开头就放弃这种加密方式。
到现在才发现我还不知道这是什么字，又在网上找的了在线汉字转拼音。得到这些汉字的读音为：

bì jí dì dà kuò hào zhè jiù shì fǔ lài gē zhí jiē jiāo lè bā dà kuò hào

意思是：bjd{这就是flag直接交了吧}，我可能比较实在就直接交了，哈哈，结果肯定不对，试了还几次，突然恍然大悟，大括号里面的应该是那个繁体字吧，结果试了一下正确了。
